# 知库 - AI Agent知识维基 - 产品规划

## 产品定位
一句话描述：为知识工作者提供 AI Agent 自动维护的个人/团队知识库，Markdown + Git 驱动，让知识自动整理而非手动管理

## 目标用户
- 主要用户群体：知识工作者、开发者、写作者、产品经理、学生
- 用户痛点：笔记越积越多但从不整理，知识散落在各处难以检索，AI需要上下文但每次都要重新粘贴
- 使用场景：日常笔记记录、项目文档管理、学习笔记整理、AI Agent长期记忆

## 核心功能
按优先级排列，最多 5 个核心功能：
1. Markdown笔记编辑器 + 文件夹组织（必须有）
2. AI自动整理分类（标签/关联/归档建议）（必须有）
3. [[wikilinks]] 双向链接 + 知识图谱可视化（必须有）
4. 草稿→正式笔记提升流（草稿区→AI审阅→正式知识库）（可选）
5. Git风格版本历史 + Markdown导出（可选）

## 变现设计
- 变现方式：Freemium + 订阅制
- 免费功能：基础笔记编辑、文件夹组织、100条笔记、[[wikilinks]]
- 付费功能：AI自动整理、知识图谱、无限笔记、Git同步、团队协作
- 定价策略：免费版够用，Pro版 19元/月，Team版 49元/月

## 技术方案
- 目标平台：Web
- 前端框架：纯 HTML + CSS + Vanilla JS（轻量快速，适合笔记场景）
- 后端方案：纯前端无需后端（localStorage + IndexedDB）
- 数据存储：localStorage（笔记内容）+ IndexedDB（大文件/版本历史）
- 第三方依赖：无（AI功能预留接口，演示模式用规则引擎）
- 部署方式：GitHub Pages
